1.工程中是不是“写函数 + 一句调用”？  
  ```text
  写法通常是这样分层的：  
      reverseList(ListNode* head) {  
      // 多行逻辑（核心算法）  
      }  
    然后在别的地方调用：  
      head = reverseList(head);  
  本质解释：用一句话调用一个已经实现好的复杂逻辑函数
  ```

2.为什么返回的是 ListNode*（看起来只是一个结点）？  
  ```text
  一个链表，本质就是“头节点的指针”
  ```

3.核心算法思想？
  ```text
  在遍历过程中，把每个结点的 next 指针指向它的前驱节点（逐个结点把指针方向反转）
  并保持链表可继续访问的状态
  ```
